# Form Validation Project

This project demonstrates **form validation** for login and signup forms using **HTML, CSS, and JavaScript**.

## Features
- Validates user input (name, email, password)
- Displays real-time error messages
- Enhances user experience with interactive forms
- Simple, beginner-friendly, and easy to customize

## Technologies Used
- HTML
- CSS
- JavaScripit
